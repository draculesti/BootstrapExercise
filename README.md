# 🍔 Burger Town

Burger Town is a simple landing page exercise created using HTML, CSS, and Bootstrap 5.  
This project was developed to practice responsive design, Bootstrap components, cards, navbar customization, alerts, hover effects, and modern UI styling.

---

## 🚀 Technologies Used

- HTML5
- CSS3
- Bootstrap 5

---

## 📚 Features

- Responsive navbar
- Hero section
- Food cards with hover effects
- Bootstrap alerts
- Modern dark theme UI
- Responsive layout using Bootstrap Grid
